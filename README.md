# dio-desafio-mentor-de-carreira-com-compilot
Este repositório contém a entrega do desafio do bootcamp da DIO, baseado no projeto digitalinnovationone/copilot-prompts (https://github.com/digitalinnovationone/copilot-prompts). O objetivo foi usar a IA Copilot para analisar minha carreira em tecnologia, considerando meu perfil e preferências, e criar um cronograma de estudos de 90 dias.

💬 ROTEIRO DE ENTREVISTAS

PERGUNTA 1: O que é HTML semântico?
COMO RESPONDER: Explique que são tags que descrevem o significado do conteúdo, como <header>, <article>, melhorando acessibilidade e SEO.

PERGUNTA 2: Qual a diferença entre var, let e const?
COMO RESPONDER: Var tem escopo de função, let e const têm escopo de bloco; const não permite reatribuição.

PERGUNTA 3: O que é o Virtual DOM no React?
COMO RESPONDER: É uma representação em memória do DOM real, que permite atualizações mais rápidas e eficientes.

PERGUNTA 4: Como você versiona seu código?
COMO RESPONDER: Uso Git e GitHub, criando branches para features e pull requests para revisão.

PERGUNTA 5: Fale sobre um projeto que você desenvolveu.
COMO RESPONDER: Estruture a resposta em contexto (problema), ação (o que fez) e resultado (impacto).
