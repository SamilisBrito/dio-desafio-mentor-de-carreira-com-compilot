# dio-desafio-mentor-de-carreira-com-compilot
Este repositório contém a entrega do desafio do bootcamp da DIO, baseado no projeto digitalinnovationone/copilot-prompts. O objetivo foi usar a IA Copilot para analisar minha carreira em tecnologia, considerando meu perfil e preferências, e criar um cronograma de estudos de 90 dias.
