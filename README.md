## 📅 Roadmap de 90 Dias
**Carga Horária:** 8 horas/semana

### Mês 1: Fundamentos
* **Semana 1-2:** Revisar HTML semântico e boas práticas de CSS; Criar páginas responsivas.

### Mês 2: Prática
* **Semana 5-6:** Introdução ao React (componentes, props e estado); Criar mini-aplicações.
* **Semana 7-8:** Consumir APIs REST com fetch/axios; Rotas e navegação no React.

### Mês 3: Portfólio e Preparação
* **Semana 9-10:** Desenvolver projeto completo (React + API); Aplicar boas práticas de UX.
* **Semana 11-12:** Publicar projeto (GitHub Pages/Vercel); Simular entrevistas técnicas.

## 💬 Roteiro de Entrevistas

> **P1: O que é HTML semântico?**
> * **Resposta:** São tags que descrevem o significado do conteúdo (`<header>`, `<article>`), melhorando acessibilidade e SEO.

> **P2: Qual a diferença entre var, let e const?**
> * **Resposta:** `var` tem escopo de função; `let` e `const` têm escopo de bloco; `const` não permite reatribuição.

> **P3: O que é o Virtual DOM no React?**
> * **Resposta:** É uma representação em memória do DOM real, permitindo atualizações mais rápidas e eficientes.
